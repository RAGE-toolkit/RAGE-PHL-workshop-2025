**Monday 24th - Friday 28th November 2026**  
*Research Institute for Tropical Medicine (RITM), Manila, Philippines*  

This workshop is a hands-on training course on the application of genome sequencing to support enhanced canine rabies virus surveillance, providing introductory training in a sample-to-sequence-to-interpretation workflow for viral pathogen surveillance, including nanopore library preparation, basic data analysis and troubleshooting masterclasses.
